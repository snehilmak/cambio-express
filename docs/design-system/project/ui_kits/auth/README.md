# Auth UI Kit — DineroBook

Split-screen login (dark navy gradient panel + cream form panel), from `templates/login.html`.

Components:
- `LoginLeft.jsx` — the navy brand panel with wordmark, tagline, and 3 feature rows
- `LoginRight.jsx` — the cream form panel with heading, email/password fields, error banner, sign-in button, signup link

`index.html` renders the pair side-by-side.
